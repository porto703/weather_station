The data file format is:
UTC_unix_timestamp meassure

The data is being stored in the following files:

temperature.txt: Temperature in ªC

relative_humidity.txt: Relative Humidity in %

pressure.txt: Air (atmospheric) pressure in Pa

dew_point.txt: Dew Point in ªC

humidity_perception: The Humidity Perception can be:
            Very dry
            Dry
            Comfortable
            Slightly humid (Getting sticky)
            Humid (Unpleasant)
            Very Humid (Oppresive)
            Dangerous
